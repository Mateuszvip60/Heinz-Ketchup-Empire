🌐 Konfiguracja JSONSilo (GitHub)

W repozytorium GitHub ustaw zmienne (Settings → Secrets and variables → Actions → Variables):

Nazwa Wartość Opis
JSONSILO_KEY TWÓJ_KLUCZ_JSONSILO Klucz API nagłówka X-SILO-KEY.
JSONSILO_FILE_ID 82305321-d682-411d-b028-83640c37fc14 UUID pliku z rankingiem.
JSONSILO_BASE_URL https://api.jsonsilo.com Adres bazowy API.
JSONSILO_REGION api Kod regionu (Niemcy: Düsseldorf).
ONLINE_FEATURES_ENABLED true Domyślne ustawienie przełącznika funkcji online.

🧩 Generowanie pliku env.js

Przed publikacją należy wygenerować plik środowiskowy. Lokalne testy mogą korzystać z kopii env.template.js. W CI/CD dodaj krok:

- name: Generuj env.js run: | cat <<'EOF' > env.js window.__ENV__ = { JSONSILO_KEY: "${{ vars.JSONSILO_KEY }}", JSONSILO_FILE_ID: "${{ vars.JSONSILO_FILE_ID }}", JSONSILO_BASE_URL: "${{ vars.JSONSILO_BASE_URL }}", JSONSILO_REGION: "${{ vars.JSONSILO_REGION }}", ONLINE_FEATURES_ENABLED: "${{ vars.ONLINE_FEATURES_ENABLED }}" }; EOF
⚠️ Pamiętaj: Plik env.js jest ignorowany przez git. Nie commituj go ręcznie – w repozytorium pozostaje jedynie env.template.js.

🚀 Uruchomienie lokalne

Krok 1: Skopiuj env.template.js do env.js i uzupełnij wartości według potrzeb.
Krok 2: Otwórz index.html w przeglądarce lub uruchom npx serve ..
Krok 3: Wprowadź swój nick (modal pojawi się przy starcie) i klikaj!
Krok 4 (opcjonalny): Dodaj obrazy do folderu skins/ i dźwięki do sounds/.

🌍 Publikacja na GitHub Pages

  1. Ustaw zmienne środowiskowe (patrz sekcja wyżej).
  2. W workflow dodaj krok generujący env.js.
  3. W Settings → Pages wybierz gałąź main i katalog /.
  4. Po wdrożeniu status rankingu w grze pokaże „JSONSilo”, a przełącznik „Funkcje online” pozwoli wyłączyć synchronizację.

🔧 Rozwiązywanie problemów

Brak synchronizacji online?
Sprawdź czy env.js zawiera prawidłowy klucz i identyfikator pliku.
Przycisk „Funkcje online” wyłączony?
Tryb online wymaga poprawnych zmiennych środowiskowych – w przeciwnym razie gra przechodzi w tryb lokalny.
Gra nie zapisuje postępu?
Upewnij się, że przeglądarka pozwala na użycie localStorage (domyślnie tak jest).
